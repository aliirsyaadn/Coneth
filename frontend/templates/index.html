Hello we are Coneth. In here we allow you to modify your personal ethernet configuration in just a matter of button click. Scroll bellow for more detail about the command that you can use





Coneth rename

Rename the device name temporary


Coneth version

Display current version

Coneth list

Display list all network and device driver


Coneth Set Device

Display help information usage


Coneth info device

Display settings information for the specific device network

Coneth infonet device

Display network information for the specific device network


Coneth down device

Down or stop running for the device network


Coneth up device

Up or start running for the device network

Coneth speed

Display internet speed
